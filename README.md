### Запуск проекта:

Для работы сайта необходим Node.js, MongoDB. Также в проекте должен быть .env файл.

<ul>
  <li> npm i в директории backend </li>
  <li> npm i в корне проекта </li>
  <li> npm run server </li>
  <li> npm start </li>
</ul>
